## TECHNICAL INTEGRATION NOTES

**Algorithm Integration:**
- **Classification Models**: Random Forest, SVM, Logistic Regression for food categorization
- **Regression Models**: Random Forest Regressor, Linear Regression for calorie prediction
- **Clustering**: K-Means for nutrient-based food grouping
- **NLP**: Text-based food logging and chatbot interface
- **Hybrid Models**: Content-based + collaborative filtering for recommendations

**Gender-Specific Features:**
- Iron-focused recommendations for women
- Protein-focused recommendations for men
- Gender-specific exercise plans
- Different calorie targets based on gender

**Philippine Context:**
- Filipino food database integration
- Local nutrition guidelines
- Cultural dietary preferences
- Traditional food classifications

**No Image Recognition:**
- Text-based food logging only
- No photo analysis features
- Manual exercise logging
- Chatbot for natural language input

**Flutter Integration:**
- Export models as pickle files (scikit-learn)
- Real-time predictions in mobile app
- Offline capability for core features
- Cloud sync for data and model updates
